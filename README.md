# 🤖 Fake AI: Камень, Ножницы, Бумага

Интерактивная фронтенд-игра "Камень, Ножницы, Бумага", написанная на чистом JavaScript (Vanilla JS). Главная фишка проекта — имитация поведения ИИ-NPC, который реагирует на исходы раундов саркастичными и ироничными репликами.

## 🚀 Особенности проекта:
* **Интерактивный интерфейс:** Реализована темная неоновая тема и плавная физика кнопок (эффекты `:hover` и `:active`).
* **Иллюзия ИИ:** Бот выбирает ходы случайным образом с помощью встроенных математических функций и отвечает уникальными фразами из баз данных (массивов) под каждый исход матча.
* **Чистый код (DRY):** Логика распределения реплик оптимизирована с помощью централизованной переменной `status`, что позволило избежать дублирования рандомизатора в коде.

## 🛠️ Технологический стек:
* **HTML5** — структурирование контента, семантические теги, DOM-структура.
* **CSS3** — Flexbox верстка, анимации, плавные переходы (`transition`), адаптивность под мобильные экраны.
* **JavaScript (ES6+)** — обработка событий (`addEventListener`), работа со случайными числами (`Math.random`), динамическое управление DOM-деревом (`textContent`), циклы ветвления (`if/else if`).
